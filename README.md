# Boxing Punch Detector
## Data set
Uses the data set found at https://www.kaggle.com/datasets/piotrstefaskiue/olympic-boxing-punch-classification-video-dataset
extract contents and place contents of zip file in raw_data

## Pretrained Weights
Pre trained weights can be found under yolo26_models and yolov8_models

## File Paths
Make sure to change the file paths for your project in yolo26training.ipynb and main.ipynb

## Toggles
main.ipynb contains the toggle sfor things such as using background subtraciton and the temporal filter at the top of the file
